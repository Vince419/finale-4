Step 1

To get PS3 games on PC you need RPCS3 (the ps3 emulator). go to https://rpcs3.net/ and download the emulator from there. you will also need to go to https://www.playstation.com/en-us/sup and download the software update for PS3. if the download says failed then hover over the failed download, press the 3 dots and press keep.

Step 2

Next you need to extract the RPCS3 file and then put the PS3 update into that extracted file. now open the RPCS3 file and then open the application and choose install settings. once installed press file, install firmware, then choose the PS3 update file you downloaded earlier. Now you can press config and set those settings as you like.

Step 3

Now to play games look up something similar to "PS3 ROMS" on a search engine and download those (make sure to extract them once downloaded). Then in RPCS3 press file, add games, and then select a folder of your extracted ROMS. From there your done and you can boot up any PS3 games you have.
